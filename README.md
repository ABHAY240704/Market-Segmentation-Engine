\# Market Segmentation \& Psychographic Analysis Engine 



\## Overview

A full-stack data science web application that performs advanced customer segmentation. It simulates demographic and psychographic customer profiles and applies machine learning to identify distinct market positions.



\## Architecture \& Tech Stack

\* \*\*Frontend UI:\*\* Streamlit, Plotly (Interactive data visualization)

\* \*\*Machine Learning Backend:\*\* Scikit-Learn

\* \*\*Techniques Used:\*\* \* Principal Component Analysis (PCA) for dimensionality reduction.

&nbsp; \* K-Means Clustering to group users based on multi-dimensional behavioral scores.

&nbsp; \* Feature Scaling (StandardScaler) for normalized distance calculations.



\## How to Run

1\. Clone the repository.

2\. Install dependencies: `pip install pandas numpy scikit-learn streamlit plotly`

3\. Launch the dashboard: `streamlit run app.py`
